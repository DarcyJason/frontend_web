# HTML 表格

## 普通表格

```html
<table>
  <!-- tr代表表格中的某行 -->
  <tr>
    <!-- th代表表格中某列标题 -->
    <th>Name</th>
    <th>Age</th>
    <th>Class</th>
  </tr>
  <tr>
    <!-- td代表表格中某块数据, 请将td中的数据与th相对应 -->
    <td>Tony</td>
    <td>18</td>
    <td>Information Security</td>
  </tr>
  <tr>
    <td>John</td>
    <td>20</td>
    <td>Computer Science</td>
  </tr>
</table>
```
