# 边框

```css
p {
  /* 点虚线边框 */
  border-style: dotted;
  /* 实线边框 */
  border-style: solid;
  /* 细实线边框 */
  border-style: groove;
  /* 短横虚线边框 */
  border-style: dashed;
  /* 双横线边框 */
  border-style: double;
  /* 浅色细实线边框 */
  border-style: ridge;
  /* 向外突出的边框 */
  border-style: outset;
  /* 向内凹陷的边框 */
  border-style: inset;
  /* 无边框 */
  border-style: none;
  /* 默认有边框的元素可以使用hidden来隐藏边框 */
  border-style: hidden;
  /* 上右下左分别设置边框样式 */
  border-style: dashed double dotted solid;
  /* 边框宽度, 其值也可以为medium, thick */
  border-width: 10px;
  /* 边框宽度, 上右下左依次设置 */
  border-width: 10px 20px 10px 20px;
  /* 简写边框宽度, 但上下相同, 左右也相同 */
  border-width: 20px 20px;
  /* 边框颜色 */
  border-color: red;
  /* 边框颜色, 上右下左依次设置 */
  borer-color: purple green yellow red;
  /* 分别设置边框样式 */
  border-top-style: dashed;
  border-right-style: dotted;
  border-bottom-style: dotted;
  border-left-style: dashed;
  /* 快速设置单边边框 */
  border-left: 10px dashed purple;
  /* 快速设置全边框 */
  border: 10px dashed red;
  /* 圆边框 */
  border-radius: 5px;
}
```
